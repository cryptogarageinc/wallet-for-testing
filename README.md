# wallet-for-testing-js

## build
npm install

## use library
```
const WalletManager = require('wallet-for-testing-js');
```

## use wallet console app
```
node wallet-console.js
```

## use single console app
- windows
```
debug_tx.bat xxxxx
```

- macos or linux
```
debug_tx.sh xxxxx
```

## test
docker-compose up
